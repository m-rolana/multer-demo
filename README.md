# Run

### Run everything
In project folder run
```
./index.sh
```

<br/>

If there no permissions run
```
chmod 655 ./index.sh
```
and then run one more time
```
./ index.sh
```
<br/>

## OR

### Run server
`node ./server/index.js`

### Run client
just open with browser `./client/index.html`
